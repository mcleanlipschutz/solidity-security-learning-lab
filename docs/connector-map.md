# Proposed Cross-System Map

This repository is the technical source of truth. Other systems should reference its evidence rather than replace it.

- **GitHub:** Contracts, tests, findings, reports, remediation, commits, CI evidence
- **Dropbox:** Private assignments, protected rubrics, diagrams, and answer keys; keep answer keys outside the public repository
- **Notion:** Curriculum notes, reflection, skills history, and recurring mistakes
- **Linear:** Exercise backlog, dependencies, priorities, blockers, and acceptance criteria
- **Google Calendar:** Approved study sessions based on actual availability
- **Gmail:** Draft weekly summaries and review communications

## Completion Relationship

Private assignment or rubric → work item → scoped exercise → proof-of-concept test → finding → remediation → regression test → report → skills update → weekly summary

A planning-system status must not override missing GitHub evidence.
