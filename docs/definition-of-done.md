# Definition of Done

An exercise may be marked complete only when all applicable criteria pass.

1. The scoped contract and exact commit are identified.
2. Intended behavior and trust assumptions are documented.
3. The finding is reproducible or the no-finding conclusion is defensible.
4. The proof of concept fails against the vulnerable version for the expected reason.
5. Impact and likelihood support the proposed severity.
6. Remediation addresses the root cause rather than only one symptom.
7. The original proof of concept no longer succeeds after remediation.
8. Normal expected behavior still passes.
9. At least one edge case, fuzz property, or invariant is tested where appropriate.
10. The written report matches the final code and tests.
11. No protected answer key was used before the first complete attempt.
12. The skills matrix cites the resulting repository evidence.
