# Exercise 01 — Community Vault Review

**Difficulty:** Foundational

## Objective

Review an Ether-holding vault with an emergency administrative function. Determine whether the authorization model remains safe when the manager interacts with other contracts.

## Prerequisites

- `msg.sender` and transaction context
- Constructors and immutable variables
- External calls
- Basic Foundry unit testing

## Scope

- `CommunityVault.sol`

## Required Deliverables

- Threat model
- Vulnerability finding or defensible no-finding conclusion
- Foundry proof of concept when applicable
- Remediated implementation
- Regression test
- Professional finding report

Do not use public solutions before submitting your first attempt.
