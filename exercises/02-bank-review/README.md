# Exercise 02 — Rewards Bank Review

**Difficulty:** Foundational to Intermediate

## Objective

Review a deposit-and-withdraw contract. Evaluate the order of operations, external-call behavior, and whether accounting remains correct under adversarial interaction.

## Prerequisites

- Contract balances and mappings
- Low-level calls
- Checks-effects-interactions
- Attacker-contract testing in Foundry

## Scope

- `RewardsBank.sol`

## Required Deliverables

- Call-flow diagram
- Vulnerability finding or defensible no-finding conclusion
- Reproducible local proof of concept when applicable
- Remediation and regression test
- Professional finding report

Do not use public solutions before submitting your first attempt.
