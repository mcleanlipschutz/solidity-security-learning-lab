# Exercise 03 — Permit Airdrop Review

**Difficulty:** Intermediate

## Objective

Review an off-chain authorization flow used to distribute Ether. Determine whether authorizations are uniquely bound to the intended claim, contract, and execution context.

## Prerequisites

- `keccak256` and ABI encoding
- `ecrecover`
- Nonces and replay protection
- Domain separation
- Foundry signature cheatcodes

## Scope

- `PermitAirdrop.sol`

## Required Deliverables

- Authorization data-flow diagram
- Vulnerability finding or defensible no-finding conclusion
- Proof-of-concept test when applicable
- Secure message-design proposal
- Remediation and regression test
- Professional finding report

Do not use public solutions before submitting your first attempt.
